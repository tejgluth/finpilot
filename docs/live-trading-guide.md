# Live Trading Guide

## Default mode

Paper trading is the default.

## Unlock path

1. Complete the 9-item risk acknowledgment.
2. Finish the minimum paper-trading period.
3. Switch broker mode intentionally.
4. Review the live unlock gate status before enabling automation.

## Operational guidance

- Start with Full Manual.
- Use small notional sizes first.
- Keep the kill switch visible and tested.
- Review the audit log regularly.
