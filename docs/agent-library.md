# Agent Library

## Analysis agents

- `fundamentals`: balance sheet, profitability, leverage, and earnings support.
- `technicals`: RSI, MACD, Bollinger Bands, moving averages, ATR.
- `sentiment`: news counts, Reddit mentions, and put/call ratio.
- `macro`: rates, inflation, unemployment, yield curve, and volatility context.
- `value`: valuation multiples, dividend yield, and filing presence.
- `momentum`: multi-horizon returns and SPY-relative strength.
- `growth`: margin quality plus earnings surprise support.

## Decision agents

- `risk_manager`: deterministic confidence and sizing gate.
- `portfolio_manager`: weighted signal aggregation plus debate review.

## Debate agents

- `bull_researcher`: strongest honest positive case from signal outputs only.
- `bear_researcher`: strongest honest negative case from signal outputs only.
